HTML5

La Estructura Web

HTML es el esqueleto de cualquier sitio web. Define párrafos, títulos, enlaces e imágenes.

<!DOCTYPE html>
<html>
<body>
  <h1>Hola Mundo</h1>
  <p>Bienvenido</p>
</body>
</html>
2

Etiquetas Esenciales

<div> - Contenedor
<a> - Enlaces
<img> - Imágenes
<button> - Botones

Atributos Clave

id - Identificador único
class - Clases CSS
href - Enlaces
src - Fuentes

✨ HTML no es un lenguaje de programación, es de marcado.
